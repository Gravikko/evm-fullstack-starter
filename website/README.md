# Website

Landing page for browsing and downloading EVM Fullstack Starter components.

## Stack

Vite + React 19 + TypeScript + Tailwind CSS 4

## Dev

```bash
npm install
npm run dev       # http://localhost:5173
npm run build     # Production build → dist/
```

## Adding Components

Edit `src/data/components.ts` — add entry with name, description, features, dependencies, chains, and code snippet.

Place ZIP archive in `public/downloads/<component-name>.zip`.
